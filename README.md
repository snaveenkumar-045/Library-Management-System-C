# Library Management System in C

## Overview
A console-based Library Management System developed using C programming and file handling.

## Features

- Login Authentication
- Add Book
- Display Books
- Search Book
- Delete Book
- Add Student
- Issue Book
- Return Book
- Display Transactions

## Technologies

- C Programming
- File Handling
- CSV Files

## Files

books.csv
students.csv
transactions.csv

## How to Run

Compile

gcc main.c -o library

Run

./library
